< ---- TOMBOL DOWNLOAD, VERSI APP, DAN RATING APLIKASI --->
Untuk menampilkan tombol download.
1.buka post lama/atau buat post baru.
2.masukin kode berikut dalam mode html, bukan compase
Kode :
<div id="Theme-details">
<ul id="download">
<li><a class="googlep" href="https://URL BLOG.blogspot.co.id/p/download.html?get_file=LINK DOWNLOAD" target="_self">Download</a></li>
</ul>
</div>
------------
Untuk menampilkan versi app
1.buka post lama/atau buat post baru.
2.masukin kode berikut dalam mode html, bukan compase
Kode : 
<div id="app-versi">
7.7
</div>

NB :  Ganti 7.7 dengan versi aplikasi yg ada

------------
Untuk menampilkan rating
1.buka post lama/atau buat post baru.
2.masukin kode berikut dalam mode html, bukan compase
Kode : 
<div id="app-rates">
<span class="stars">4.6</span></div>
</div>

NB : Ganti 4.6 dengan rating aplikasi tersebut atau bisa terserah anda
